Aerospace-modeling-simulation

Developing codes & learning scientific computing to solve common problems in Aerospace Engineering using Python, a very good free alternative. An online community where we can collaborate and create python codes specific to different areas of aerospace engineering like Control and Guidance laws, CFD, FEM etc. using libraries like numpy, scipy and matplotlib. The intention is NOT to create a standalone package or to commercialize it, we could deal with specific problems and submit our little codes for comparing it with each other. This can help us learn more about mathematical modeling and application of therotical concepts. Please feel free to contribute, participate, learn and grow!

Please go to "Branches" and select an approprite topic to view and contribute code. Contributors can also submit their code from a different topic and an appropriate branch will be created for the same.

You may require Python 3, NumPy, Matplotlib to run codes. No code uses SciPy as of now but it might be required later.

Contributors are free to use any additional module as long as it is freely available and compatible with Python 3.
